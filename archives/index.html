排序算法之希尔排序

发表于 2015-03-28   |   分类于 算法   |  

希尔排序，也称递减增量排序算法是插入排序的一种更高效的改进版本。希尔排序是非稳定排序算法。希尔排序通过将比较的全部元素分为几个区域来提升插入排序的性能。这样可以让一个元素可以一次性地朝最终位置前进一大步。然后算法再取越来越小的步长进行排序，算法的最后一步就是普通的插入排序，但是到了这步，需排序的数据几乎是已排好的了（此时插入排序较快）。[1]

希尔排序是基于插入排序的以下两点性质而提出改进方法的：

  • 插入排序在对几乎已经排好序的数据操作时，效率高，即可以达到线性排序的效率
  • 但插入排序一般来说是低效的，因为插入排序每次只能将数据移动一位

[2]

理解了插入排序，再理解希尔排序就容易多了。可详见我的上一篇文章排序算法之插入算法。

排序算法之插入排序

发表于 2015-03-28   |   分类于 算法   |  

插入排序（Insertion Sort）是一种简单直观的排序算法。它的工作原理是通过构建有序序列，对于未排序数据，在已排序序列中从后向前扫描，找到相应位置并插入。插入排序在实现上，通常采用in-place排序（即只需用到O(1)的额外空间的排序），因而在从后向前扫描过程中，需要反复把已排序元素逐步向后挪位，为最新元素提供插入空间。[1]

[2]

先来看插入排序的步骤：

  • 首先假设第i个元素已经有序；

  • 第i+1个原始抽取出来作为参考，于是i+1位置可看作空位；

  • 把第i+1个元素左边的元素（即第1到i个元素）依次从右向左与第i+1个元素相比较，若比i+1大，向后挪一个位置；

  • 上面排完之后数列中还剩一个位置，把之前抽取出的第i个元素补上。

  • i将数组循环遍历，重复以上过程。

下面用实例来说明。

原始数列为：

|3|5|2|4|7|9|8|1|6

|-|

我们假设3为有序数列，已经排好。所以从5开始。只需要看3和5两个数。5抽取出来，由于3比5小，数列不变。

接下来3、5有序，只需看3、5、2三个数。2抽取出来，由于5比2大，所以5向右挪一个位置，即覆盖2刚刚的位置；3比2大，3向右挪一个位置，即覆盖最初5所在的位置；2补上空位，即第一个位置。得：

|2|3|5|4|7|9|8|1|6

|————|————|

同理，继续下去，4抽取出来，只需看2、3、5、4四个数。得：

|2|3|4|5|7|9|8|1|6

|————|————|

下面来分析插入算法的时间复杂度：

  • 最坏的情况，是逆序。比较次数O(n2),移到次数也是O(n2)，所以时间复杂度是O(n2)。

  • 最好的情况，正序。比较次数O(n)，交换次数为0，所以时间复杂度是O(n)。

  • 插入排序不适合对于数据量比较大的排序应用。但是，如果需要排序的数据量很小，例如，量级小于千，那么插入排序还是一个不错的选择。 插入排序在工业级库中也有着广泛的应用，在STL的sort算法和stdlib的qsort算法中，都将插入排序作为快速排序的补充，用于少量元素的排序（通常为8个或以下）。[3]

排序算法之选择排序

发表于 2015-03-27   |   分类于 算法   |  

选择排序（Selection sort）是一种简单直观的排序算法。它的工作原理如下。首先在未排序序列中找到最小（大）元素，存放到排序序列的起始位置，然后，再从剩余未排序元素中继续寻找最小（大）元素，然后放到已排序序列的末尾。以此类推，直到所有元素均排序完毕。[1]

很明显，选择排序的算法过程是：

  • 假定第一个元素最小；
  • 从左到右循环数组，不断将最小值复制给最小的数；
  • 遍历的每一步结果，都是当前数组中最小的数移到当前数组的最前面。

分析选择排序的时间复杂度：

  • 最坏的情况，逆序下，两个for循环不断去比较,比较次数是0(n2)，交换次数是O(n),因此时间复杂度是0(n2);
  • 最好的情况，正序下，仍然不断去比较，比较次数O(n2)，交换次数为0，因此时间复杂度是0(n2)。
  • 交换所需的时间比比较多，所以n不大时，一般说来，选择排序性能优于冒泡排序。

排序算法之冒泡排序

发表于 2015-03-27   |   分类于 算法   |  

冒泡排序（Bubble Sort）是一种简单的排序算法。它重复地走访过要排序的数列，一次比较两个元素，如果他们的顺序错误就把他们交换过来。走访数列的工作是重复地进行直到没有再需要交换，也就是说该数列已经排序完成。这个算法的名字由来是因为越小的元素会经由交换慢慢“浮”到数列的顶端。

华为2015年武汉地区技术服务工程师面试offer

发表于 2014-10-25   |   分类于 面试   |  

在保研成绩还没出来之前，去了一趟华为的面试，投的岗位是技术服务工程师。十月份已拿到offer，但我已保研去复旦。在此记录一下整个面试经过，以供后来的同学参考吧。

我参加的是华为的提前一批校招，当时在武汉的华科、武大、理工相应的学院进行了内部的宣讲，但是微博华为武汉长沙校园招聘上也会有相应通知。

当时了解到本科如果不是做过研发相关项目或者已在华为参加过实习的话，相关研发岗位对于本科生来说还是比较困难的，所以选择了技术服务工程师。关于这个岗位的好坏，众说纷纭，我获取的相关信息大多来自于天涯论坛华为板块。在我看来，虽然是辛苦，但是到手的钱多，在国外也算是一番历练，所以对于不怕吃苦的学生来说，这个岗位还是具有吸引力的。

下面详细说一下整个面试流程吧：

1.九月初华为来学院宣讲后，我就在华为官网上填写了相关简历信息。

2.16号收到华为17号在洪广大酒店的面试通知。在这之前没有一点征兆。所以没有收到短信的同学不要着急，可以问下其他同批次的同学，因为很多面试通知都是提前一天发的。

3.17号早上八点赶往洪广大酒店。因为技术服务工程师是非研发岗，所以第一面是传说中的群面。在去面试之前我在应届生论坛上看了很多面经，做好了充足的准备，在此也推荐下这个网站应届生论坛。群面有16人，面试官3人。首先每个人进行自我介绍，面试官看个人介绍的情况可能会随机追问，最重要的是在一分钟内让面试官和其他人记住你，切记，千万不要超时，面试官对时间卡的真严。然后分为两组，抽到的题目是“知易行难”和“知难行易”。事先选出leader，timer等，然后再进行小组讨论，时间为五分钟。最后由leader进行总结阐述。每个环节都有时间限制，切记不可超时。然后两组分别就己方和对方辩题进行辩论。面试官会随机对某人进行提问。其实，在我看来，群面无非就是考查团队合作以及表达能力。对结果并不在乎。只要小组内协调一致，而你又能发出自己的声音，基本上不会有问题。淘汰的往往是那些不开口，和团队意见不一致的人。比如我们的leader，当面试官问他我们小组贡献率最低的是谁的时候，他沉默了很久，然后说每个人都有贡献，面试官让他一定要说出一个人，他想了很久，最好说了自己。而前面有几位组员任务他的贡献率最大。最后，这个leader被淘汰了。

4.群面之后就是技术面。技术面是单面，一个面试官拿着电脑随机问问题，我就是按照聊天的形式回答的。该笑笑的时候笑笑，该严肃的时候严肃，该写写的时候写写。主要是针对简历里的情况进行问答，所以建议提前把简历的所以情况都心中有数。最后，会问到专业科目的问题，然后根据你的证书或者科目随机问一两个题目。比如，问到我的是给出一些主机的子网掩码。

5.技术面之后就是boss面。但在boss面之前会要求昨晚性格测试盒英语测试。性格测试一定要按照第一感觉填写，不然你真的可能在做完之后觉得自己性格分裂了。英语测试分为选择和口试，选择部分不难，过了四级的应该都没问题，口试就有难度了，随机给出一个题目，然后让你阐述一下。之前宣讲的时候说今年可能没有英语测试了，所以没准备，不过也坎坎坷坷过去了。昨晚测试后，接近中午了。工作人员让我们吃完饭下午再来。

6.下午Boss面真正是聊天形式了，聊聊学校。聊聊家庭，聊聊感兴趣的事儿。比较轻松。最后会问，“你有什么问题想问的”。建议事先想一想一两个问题，最好跟自己的岗位和公司相关，以表明自己对这个职位的深思熟虑。

7.聊完了之后，一天的面试就结束了。让回去等通知。知道华为从长沙回来的第二天，武汉地区发offer的第一天，也就是10月16号，我收到了offer。所以建议密切关注华为的招聘微博。

当然，我拒掉了。这也算是我面试挖到的第一桶经验吧。发现面试也不是很难，华为传说中吓人的群面也不过如此，不必惊慌，多看面经心里准备好一般都没问题。

最后，祝所有找工作的同学都拿到心仪的offer。

复旦电子工程系推免复试经验（外校）

复旦电子工程系推免复试经验（外校）

发表于 2014-10-08   |   分类于 面试   |  

2015年是教育部研究生推免改革第一年，我本来成绩不是很好，不过借着这个改革的春风，参加了复旦电子工程系的电路与系统专业的面试。在去上海之前，我在网上搜索了一下相关关键词，发现没有什么可以具体参考的经验。于是在面试结束后，决定写下这篇文章，权当对自己的肯定，也希望对后来者提供一定的参考价值。

以下所说，仅仅只针对复旦大学信院电子工程系的复试。

1.复旦的复试，在所有有名的高校里面算是比较晚的了，今年是10月7日，据我所知，其他的名校之前已经结束了复试。所以保外的同学可以先去其他学校看看，然后再来复旦，时间不冲突。

2.复旦老师说，在推免系统上淘汰了部分学生，具体真实性无从得知。参加复试的那一天一共来了14个学生，当时是生物医学和电路与系统同时招生，一共招8个推免生，生物医学2个，电路与系统6个，其中生物医学和电路与系统一边一个直博。从概率上说，进去的概率还是比较大的。

3.复试题目：一道英文题和一道中文专业题。会提前让每一个人单独在一间房间里准备15分钟。英文字数不多，大概150个单词，但是涉及的内容仍然与专业相关，会碰到一些生词，可用手机查。个人建议查清楚英文所阐述的那个专业知识点是什么，比如我当时抽到的题目是PID，最好就百度谷歌一下PID是干嘛的。中文专业知识点则涉及方方面面，从模电数电单片机到信号与系统，都有可能涉及。所以最好提前把这些书全都过一遍，有时间的话最好仔细把原理什么的弄懂。

4.面试环节，与其他很多学校都不一样，不需要自我介绍，没有一句话是闲聊的话，全都是涉及专业知识的问题，而且涉及的知识非常细致，会让你把具体原理甚至电路画出来。而且，直博和硕士貌似还不一样。听前面那直博哥们说，每个老师都问了问题，似乎是一种意义上的压力面试，出来的时候他说他整个人都不好了，不好了。进去发现，大概有十个老师，不需要递交简历什么的，也不需要自我介绍。先让把英文读一句翻译一句，读完了翻译完了，可能会问一两句英文涉及的知识点的问题。然后要求把中文问题解答一下，解答过程中会问各中非常细致的问题，这就要看之前复习的怎么样了。当然，肯定有一些没看到或者疏忽了，建议不要瞎扯，不记得的千万不要乱说，可以诚实地说没复习到或者忘记了。回答过程中不要紧张，把会的全都答出来就好。因为面试的同学都不可能全部答出来，尽量把自己的状态展示出来最好了。

5.当天下午就出结果了。

6.建议在去复旦之前联系一个老师，只要老师没明确拒绝或者没回邮件，都表示有希望。面试结束后，尽早联系那个老师，达成双向选择后，基本上就高枕无忧了。武大有个朋友，因为没早点联系老师，心仪的老师名额已满，拖了一天才确认好老师。
